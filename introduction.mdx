## What is GLOMAP?

GLOMAP is a general-purpose **global Structure-from-Motion (SfM)** pipeline for image-based 3D reconstruction. Unlike incremental SfM approaches that register images one at a time, GLOMAP solves for all camera poses simultaneously using global rotation averaging and global positioning — making it dramatically faster while maintaining reconstruction quality.

Key properties:

- Takes a **COLMAP database** as input (feature extraction and matching handled by COLMAP)
- Outputs a **COLMAP sparse reconstruction** — fully compatible with downstream tools
- Typically **10–100× faster** than COLMAP's incremental mapper
- On-par or superior reconstruction quality compared to incremental SfM

## How it works

GLOMAP runs a modular pipeline with the following stages:

<Steps>
  <Step title="View graph calibration">
    Calibrates the view graph using camera intrinsics from the COLMAP database.
  </Step>
  <Step title="Relative pose estimation">
    Estimates pairwise relative poses between image pairs using RANSAC-based essential/fundamental matrix estimation via PoseLib.
  </Step>
  <Step title="Global rotation averaging">
    Solves for all absolute rotations simultaneously using global rotation averaging (optionally with a gravity prior for 1-DoF rotation).
  </Step>
  <Step title="Track establishment">
    Builds feature tracks across images from pairwise correspondences.
  </Step>
  <Step title="Global positioning">
    Solves for all camera positions simultaneously using the estimated rotations and track observations.
  </Step>
  <Step title="Bundle adjustment">
    Jointly refines all camera poses and 3D point positions using Ceres Solver (with optional GPU acceleration).
  </Step>
  <Step title="Retriangulation">
    Re-triangulates 3D points to improve the final reconstruction density.
  </Step>
</Steps>

## Relationship to COLMAP

GLOMAP is designed to work **alongside COLMAP**, not replace it entirely:

| Task | Tool |
|------|------|
| Feature extraction | `colmap feature_extractor` |
| Feature matching | `colmap exhaustive_matcher` / `sequential_matcher` / `vocab_tree_matcher` |
| **Global mapping** | **`glomap mapper`** |
| Visualization | `colmap gui` |
| Dense reconstruction | `colmap patch_match_stereo` |
